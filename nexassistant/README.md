# Nex-Assistent

> Dokumenten-Assistent mit Gedächtnis für KMU

## Das Problem

KMU ertrinken in:
- Briefe, Rechnungen, Angebote
- E-Mails mit Anhängen
- Zettelwirtschaft auf dem Schreibtisch
- "Wo war nochmal dieser Brief von Januar?"
- Fristen vergessen (Zahlungsziele, Verträge, Kündigungsfristen)

**Pain Level:** Hoch - jeden Tag, jeden Moment

---

## Die Lösung

Dokumenten-Assistent mit Gedächtnis:

1. **Foto eines Briefs** → App oder WhatsApp
2. **E-Mail weiterleiten** → An dokument@nex-assistent.de
3. **Chat-Upload** → Natürliche Sprache

---

## Auto-Magic

### 1. Kategorisierung
- Rechnung / Angebot / Vertrag / Mahnung / Behörde
- Absender erkannt
- Betreff extrahiert
- Betrag & Datum

### 2. Fristen-Erkennung ⏰
- Zahlungsziel: "Zahlbar bis 15.03."
- Kündigungsfrist: "Kündigung 3 Monate zum Quartalsende"
- Termin: "Bitte kontaktieren Sie uns bis..."

### 3. Vernetzung 🔗
- "Diese Rechnung gehört zu Angebot #1234"
- "Ähnlicher Brief von selben Absender"
- "Vertrag XY läuft aus - Erinnerung!"

### 4. Vektor-Brain 🧠
- Alles durchsuchbar
- "Zeig mir alle Rechnungen über 500€"
- "Was habe ich von Firma XY?"
- "Welche Fristen laufen diese Woche?"

---

## Unterschied zu NexWorker

| NexWorker | Nex-Assistent |
|-----------|---------------|
| Baustelle / Field Service | Büro / Admin |
| Monteure dokumentieren vor Ort | Geschäftsführung / Bürokräfte |
| Fotos, Sprachmemos | Briefe, E-Mails, Rechnungen |
| Extern | Intern |
| Workflow-Doku | Dokumenten-Intelligenz |

---

## Zielgruppe

1. **Handwerksbetriebe** (10-50 Mitarbeiter)
   - "Ich bekomme 20 Briefe am Tag"
   - "Meine Sekretärin ist überlastet"

2. **Freiberufler** (Steuerberater, Anwälte)
   - Hohe Dokumentenlast
   - Fristen kritisch

3. **KMU im Allgemeinen**
   - Jeder mit Papierchaos
